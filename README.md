# 🎵 Musice Tune

Musice Tune is a simple and modern Spotify Clone built using HTML, CSS, and JavaScript. It provides an interactive music player interface where users can play, pause, and navigate songs easily.

---

## 🚀 Features

*  Play and Pause songs
*  Next and Previous controls
*  Clean and interactive UI
*  Album cover display
* Music progress bar
* Fast and lightweight

---

##  Technologies Used

* HTML5
* CSS3
* JavaScript

---

## How to Run

1. Download or clone this repository
2. Open the project folder
3. Run `index.html` in your browser
4. Enjoy the music player 

---

##  Purpose

This project was created for learning web development and practicing JavaScript, DOM manipulation, and UI design.
